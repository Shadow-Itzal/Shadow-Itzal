### 🌱 Sobre mí

Soy **desarrolladora en formación** con foco en **Frontend JavaScript y React**, además de experiencia en **Diseño UX/UI**.

Me apasiona crear soluciones que integren diseño, funcionalidad y experiencia de usuario, transformando ideas en productos digitales intuitivos y atractivos.

Actualmente me encuentro profundizando mis conocimientos en **React**, fortaleciendo mis habilidades de desarrollo frontend moderno, y perfeccionándome en **Inteligencia Artificial**, explorando herramientas y tecnologías aplicadas al desarrollo de software.

También cuento con conocimientos en:

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- MongoDB
- SQL
- Figma
- Git y GitHub

Además, me he formado en habilidades blandas que me permiten trabajar de forma colaborativa, adaptarme a nuevos desafíos y mantener una actitud de aprendizaje continuo.

### 📌 Proyectos destacados

- **[Bruma Galería](https://github.com/Shadow-Itzal/Bruma)**  
    E-commerce de arte premium desarrollado con **React, Vite, React Router, Context API y CSS Modules**. Incluye catálogo dinámico, carrito global, panel administrativo e integración con ImgBB para carga de imágenes.
  🔗 https://dominguezmarta-bruma.netlify.app/

- **[Bizitzal](https://github.com/Shadow-Itzal/Bizitzal)**  
    E-commerce de kits creativos desarrollado con **HTML, CSS y JavaScript**, utilizando renderizado dinámico desde archivos JSON, carrito persistente con localStorage y diseño responsive.
  🔗 https://shadow-itzal.github.io/Bizitzal/

- **[TrailerFlix](https://github.com/Shadow-Itzal/TrailerFlix_Final)**  
    API REST construida con **Node.js, Express y MongoDB**, permitiendo consultar información de películas y series desde un dataset estructurado.

- **[Comida Sana, Vida Sana](https://github.com/Shadow-Itzal/Grupo-04)**  
    Proyecto colaborativo de desarrollo web realizado con **HTML, CSS y JavaScript**, aplicando metodologías ágiles y control de versiones con Git.
  🔗 https://shadow-itzal.github.io/Grupo-04/

- **[Carolina Silva - Fotógrafa de bodas](https://github.com/Shadow-Itzal/G19-fotografia)**  
    Sitio web responsive desarrollado con **HTML, CSS, JavaScript y Python**, enfocado en experiencia visual y presentación profesional de servicios fotográficos.
